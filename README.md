# AlgoViz: Path-finding & Sorting Algorithms Visualizer

AlgoViz is an interactive application that demonstrates pathfinding and sorting algorithms through animations.

## Key Features

- Pathfinding algorithms: A-Star, Dijkstra, DFS, BFS
- Sorting algorithms: Quick-sort, Merge-sort, Heap-sort, Radix-sort, and more
- Dynamic pathfinding with midpoints and mouse/finger chase
- Visualization controls: Pause, Play, Speed adjustment, Player-Bar
- Mobile-responsive design
- Dark mode
- Code implementation in JavaScript

## Built With

React, Redux, and Material-UI
